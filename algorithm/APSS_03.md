### 2. 일관되지 않은 범위 표현방식
* 범위 표현방식의 혼동을 피하려면 프로그래밍 언어에서 지정한 범위 표현 방식 사용
* [a, b) 사용
* `for (int i = a; i < b; i++)`, 다른 `for (int i = a; i <= b; i++)`과 같은 범위와 혼용하다보면 헷갈릴 여지가 많다

### 너무 큰 중간값

### 무한대
* 무한히 큰 값을 설정할 때 오버플로로 인한 음수값이 발생하지 않고 적당히 크면서 정수 범위에 들어서는 값을 사용
* `int max = 987654321`
